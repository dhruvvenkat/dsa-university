## Additional Topics (Future Learning)

These topics are mentioned but not covered in the 30-day schedule. Consider learning them after completing the core curriculum:

- **Dynamic Programming:** Start with Fibonacci, coin change, longest common subsequence
- **Greedy Algorithms:** Activity selection, fractional knapsack, interval scheduling
- **Backtracking:** N-queens, sudoku solver, subset generation
- **Union-Find:** Disjoint set union, connected components, Kruskal's algorithm

## Study Tips

1. **Time Management:** Break 2 hours into focused sessions (e.g., 45 min study, 15 min break, 45 min practice, 15 min review)

2. **Active Learning:** 
   - Write code yourself (don't just read)
   - Explain concepts out loud
   - Draw diagrams for complex structures

3. **Problem-Solving Process:**
   - Understand the problem
   - Identify patterns
   - Plan approach
   - Implement
   - Test edge cases
   - Analyze complexity

4. **Review Strategy:**
   - Review previous day's concepts each morning
   - Keep a problem log
   - Note patterns and insights

5. **Practice:**
   - Solve problems without looking at solutions first
   - Try multiple approaches
   - Focus on understanding, not memorization
